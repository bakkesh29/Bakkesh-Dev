> 🎓 Final Year Project | Information Science Engineering | VTU | 2025-26

🌐 **Live Demo:** https://bakkesh-dev.vercel.app
📦 **Backend API:** https://bakkesh-dev-backend.onrender.com

## About
Full-stack MERN portfolio website showcasing projects, achievements, skills, and daily developer activity with JWT authentication and MongoDB Atlas.

## Tech Stack
- **Frontend:** React, Tailwind CSS, React Router, Axios
- **Backend:** Node.js, Express.js, MongoDB, JWT, Nodemailer
- **Deployed:** Vercel (Frontend) + Render (Backend) + MongoDB Atlas

## Run Locally
```bash
git clone https://github.com/bakkesh29/Bakkesh-Dev.git
cd Bakkesh-Dev/server && npm install && npm run dev
cd ../client && npm install && npm start
```
